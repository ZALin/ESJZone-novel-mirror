20XX年，一款世紀MMORPG遊戲破鏡而出，然後...我還沒想好


前言:世界樹的誕生X由來
[前言:世界樹的誕生X由來] by Alive Me

<System World>系統世界，乃是人類至今算得上是數一數二的世紀大發明。能使人類的意識脫離本體，暢遊在萬象大千的網路世界。
<System World>創造了許多人夢寐以求的世界，虛擬潛行裝置可說是幾乎辦到了人類所能做的限制。
林林種種的遊戲類型出世，讓人們生活不再那麼的枯燥乏味。甚至可以說是拯救了某些人。
而如今，自<System World>出現至今已經過了五年，<System World>不再像當初那樣的雜亂無章，超過百萬種的遊戲經過了五年的時間，早已沒了當初的人氣，只因沒有那些精序的代碼工程，導致了許多遊戲時的不順暢。甚至跌落神壇的遊戲不在少數。然而有一款遊戲，它奠定了<System World>中第一遊戲的地位，至今無人可撼動。
<Tree System >是<System World>出世後一年才發布的MMORPG遊戲，可說來也奇怪。在出發點就已經落後的這款遊戲，就連發佈後也沒有大肆宣傳，只是靜靜地等待著人們前來遊玩。明明是如此的低調，卻成了現在無可撼動的一座大山。
然而直到現在，所有有關製作商的消息卻是一條都沒有，沒錯，任何一條。簡直就像自行管理的系統一般，讓人如此的匪夷所思。
盡管如此，還是擁有許多的人們遊玩，而這是遊玩過的人們所給出的感想:無法想像它只是遊戲。
<Tree System >有著魔法，特性，技能等所有能想到的RPG遊戲都有的系統，然而有一點正是<Tree System >佔據第一的原因。
就如同它的名字是樹系統一樣，這款遊戲的劇情就像是樹一樣，擁有無限的分支就擁有無限的劇情，四年時間還是沒有人能完全破關遊戲，甚至是讓人們感覺到一種「無論如何都無法通關」感覺，這劇情是如此之多，如此之大，就猶如世界一般。
因此<Tree system >也有另一個外號，<World Tree System >世界樹系統，無止境的劇情分支延伸至世界，世界的無窮遠不是人類所能觸及的，才因此被稱作<World Tree System >
而且明明遊戲是如此之大，所有的細節卻能做到完美，明明是NPC卻能像真人一般的說話活動，簡直完全看不出缺點。
盡管< System World>是創世紀大發明，然而這款遊戲的製作卻遠比這項發明要來的恐怖，無人能得知這款遊戲是輸入甚麼程式代碼，是如何讓它變成如此的真實，甚至連如何製作都不知道，數年來許多人駭入系統來盜取資料，然而卻彷彿有股無形的牆壁阻擋一樣，時至今日沒有任何一人能探知其中的秘密，因此這也被當作是一個都市傳說。
這就是<World Tree System >
無可觸及的世界樹。
 

01.為虛為幻的慾望
[01.為虛為幻的慾望] by Alive Me

『慾望』是人們心底最原始的情感。
人都擁有想要的事物，為達目的不惜一切。
慾望為母，慾望就是源源不斷動力來源，支撐著需要慾望來達成的一切目的。
然而，接觸不到的慾望，就是虛幻的。為虛為幻，亦為不存在。
人類會為了不存在的事務而感到憧憬，會不惜一切代價得到想要的虛幻。而慾望則就是為了此刻存在的，因此虛幻的事物也被稱為『都市傳說』
—
「你給我出去，我沒有你這個兒子。」
「別這樣...老公，聽孩子解釋阿...」
「從今以後我沒有你這個兒子。」
玻璃的碎裂聲響起，父親暴怒的將我的物品全部都摔了出去，對於平時只要聽母親說幾句氣就會消停的父親來說，想必我很讓他失望吧。
我從小便很憧憬女性，對於男生的東西我都不感興趣。洋娃娃、裙子、化妝品，到最後甚至連『想要變成女生』的念頭，漸漸地...變成了願望。
靈魂活在錯誤的身體裡讓我非常的痛苦...甚至讓我擁有想要自殺的念頭。
...好痛苦，真的好痛苦。
我很討厭我這副身體，然而我卻無法做出任何表態，只因父親是個很古板的人。
我從未像現在一樣感到自己是多麼的弱小，甚至連最簡單的表態都做不出。
然而那年...<System World> 出世後，我第一次感覺自己是離願望是多麼的近，甚至觸手可及。
我以女性身姿、容貌、聲音，開始遊玩潛行遊戲，我的心情從未達到過如此的興奮。
男生的汗味變成了女生獨有的體香、不再擁有粗獷的身體，取而代之的是吹彈可破的肌膚與女生嬌小的身軀。
不再擁有沖破天際的棒狀物而是變成了令人垂涎的洗面乳和吸引萬千男生的粉紅鮑廂。
阿...真香~

『現在開始深入教程。』
一道讓人身子酥麻的撫媚聲音傳來，想必那就是這個遊戲的引導NPC了。

然後我的身體(虛擬)便開始了自我運動…

「嗯…阿…呼呼。」
淫蕩的嬌喘聲迴盪在空氣中。
我的手不受控的開始撫摸自己，一隻手揉著胸器、玩弄著自己的乳頭，身子彷彿像是觸電了一樣，完全使不上任何力氣。
「…這就是…女性的身體…嗎?」

好舒服，這完全不是男性的身體可比擬的。
此時，另一隻手卻開始慢慢地往下撫摸，慢慢的…慢慢地。

「阿。」
在手指觸碰到荳荳的那刻，一股從身體深處猶然而生的興奮感湧起，此刻腦子完全空白，整個身心都沉溺在這份快感之中，無法自拔。
此刻手指不自主地開始加快力道與速度，如果說先前的速度只是前菜的話，現在就是開始品嘗主菜的部分。
手指自動伸入了蜜穴的深處，彷彿堪比金手指加藤桑的手部技巧，淫蕩的水聲在蜜穴裡滋滋作響，在興奮感達到極致的那個瞬間，彷彿無法感受到任何事物一般，只沉溺在那無盡的輪迴之中…

而五年後的今天，也就是我下定決心向父親說明變性的事情。
然而父親卻堅決不同意，我從未看過父親如此的生氣，在他把所有的玻璃製品摔出的時候，我的心…便隨著它一起破碎了…

徬徨無措的我走在街道上，像是映襯我的心情一樣，此刻大雨滂沱，順著臉頰所滴下的眼淚甚至在一瞬間就和雨滴融合了。
身處在絕望谷底的我看像了谷頂，像是要尋找那道微小希望一般，就像死命抓著救命稻索的人一樣，我抬頭看著天空，意識…卻在此刻消逝無蹤。 

02.陌生之人
[02.陌生之人] by Alive Me

「處理好了嗎？」
空洞、虛無、縹緲，從黑暗的深處傳來的聲音，那是令人打從心底感到冷顫的聲音，一不注意就彷彿會被吞噬殆盡一樣。
恐怖不已，光是聲音就能令人感到如此，能散發出這種感覺...絕對不是人類。
「嗯，畢竟是欠你的人情，沒事的話我就先走了。喔，對了，我很期待那天的到來。」
「沒事的話就走吧，無論發生什麼事，在她回來之前，我都不會出手。」
如果說深處傳來的聲音是虛無縹緲的，那麼此刻令一位則是給人剛勇堅毅的感覺，就像傲然矗立的山峰一樣，永垂不朽。
從兩人之間的談話中不難看出，他們口中的她，應該是對兩人來說很重要的存在吧。
與此同時，某處消失的意識再次醒來…

—

「吶。」
究竟活著還有什麼意義呢？
如果說錯誤的靈魂活在了錯誤的身體裡，強行驅使著不合零件的器械，終有一日還是會毀壞。人亦是如此，雖說我生為男生，但追求女性又有何不對。
一昧的阻擋人的希望，只會讓人活下去的動力越來越小。
「吶。」
為何父親不同意我的想法，真正讓我完成心願又如何。他會損失什麼嗎？錢？面子？這有比他們唯一一個孩子內心的想法珍貴嗎？
雖說沒有父母親就沒有我，但…真的破壞了一個人活下去的希望，那麼…就如同行屍走肉般…令人唏噓。不如死了才是解脫。
「吶！我說！！！」
咦？
因為突如奇來的聲音而打斷了我的思考，話說，我連有沒有人在旁邊都不知道。
「妳終於注意到我了，在想什麼呢？想的那麼入迷。」
「沒有，沒什麼。」
幾乎是下意識的說出，雖說不知為何要否認就是了。
「那個，妳是誰？為什麼在這裡？」
好美麗…該說是氣質嗎？看了她彷彿就會被吸引一樣，有種無形的吸引力在牽扯著我，好似一不注意便會愛上她一樣。
「啊，討厭，可別愛上我阿，不過我可不討厭自己愛上自己。」
自己愛上自己，什麼意思？
話說，她為什麼會知道我在想什麼？
然而就在我的大腦還沒來得及運轉時。不如說，我連自己在哪都不知道，連她是誰都不知道，我的意識又再次消散了。
「不過…是哪裡出錯了嗎？我的最後一個人格…算了，總會恢復的，或許…這次會有不一樣的結果呢。」
留下這麼一句令人不解的話，人就消失了，聲音飄散在空氣中，漸行漸遠。
